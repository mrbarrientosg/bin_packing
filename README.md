# Bin Packing con Basic VNS

Para compilar el codigo debe ejecutar el siguiente comando de linea:
```bash
$ cd bin_packing
$ g++ -O3 *.cpp -o main --std=c++11
```

Para ejecutar:
```bash
$ ./main ../instances/HARD0.BPP
```
